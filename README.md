## 成都修茈科技有限公司 · 企业门户网站

这是一个为 **成都修茈科技有限公司** 搭建的简洁现代的公司门户网站雏形，采用纯 HTML + CSS + JavaScript 实现，方便直接部署到任意静态服务器或对象存储（如 Nginx、IIS、GitHub Pages、OSS、COS 等）。

### 功能区块

- **首页（Hero）**：公司价值主张与核心服务简介。
- **产品与服务**：展示定制开发、移动端/小程序、技术咨询与运维等服务模块。
- **行业解决方案**：三个示例行业方案，可根据实际业务调整文案。
- **关于我们**：公司简介、团队优势与关键指标展示。
- **联系我们**：联系方式信息与一个演示用的联系表单（前端提示，不真实发送）。

### 运行与预览

1. 使用浏览器直接打开 `index.html` 即可预览。
2. 如需本地起一个简单的静态服务（可选）：

   ```bash
   # 方式一：如果安装了 Python
   cd "E:\成都修茈科技有限公司"
   python -m http.server 8080
   # 然后浏览器访问：http://localhost:8080

   # 方式二：Node.js 用户可使用任意静态服务工具（如 serve、http-server 等）
   ```
