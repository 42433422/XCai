## 一、就业形势概览

### 1.1 招聘规模

| 年份 | 招聘类型 | 招聘人数 | 备注 |
|------|---------|---------|------|
| 2025年 | 成都公招 | 948名 | 编制内中小学教师 |
| 2025年 | "蓉漂人才荟" | 2306名 | 各区县赴高校考核招聘 |
| 常年 | 员额教师 | 不定 | 非编制，有机会转编 |

### 1.2 历史学科需求特点

- **需求稳定**：历史作为中考、高考主干学科，各学校均有固定编制需求
- **竞争激烈**：历史专业毕业生较多，主城区岗位竞争较大
- **区县机会多**：二三圈层区县及地市州历史教师缺口相对较大

## 二、成都地区招聘详情

### 2.1 招聘类型对比

| 招聘类型 | 编制情况 | 学历要求 | 考试方式 | 竞争程度 |
|---------|---------|---------|---------|---------|
| **公招考试** | 事业编制 | 本科及以上 | 笔试+面试 | ⭐⭐⭐⭐ |
| **蓉漂人才荟** | 事业编制 | 硕士为主 | 直接面试 | ⭐⭐⭐ |
| **员额教师** | 无编制 | 本科及以上 | 学校自主招聘 | ⭐⭐ |
| **两自一包** | 无编制 | 本科及以上 | 学校自主招聘 | ⭐⭐ |

### 2.2 学历要求详解

#### 市属/主城区高中
- **基本要求**：硕士及以上学历
- **优先条件**：985/211院校、部属师范院校
- **放宽条件**：特别优秀者可放宽至本科学历
- **证书要求**：2025年7月31日前取得毕业证、学位证、教师资格证

#### 区县高中
- **基本要求**：本科及以上学历并取得相应学位
- **专业要求**：所学专业与应聘岗位相符或相近
- **年龄限制**：一般要求1989年8月1日之后出生（35周岁以下）

#### 初中岗位
- **基本要求**：本科及以上学历
- **专业要求**：历史学类、学科教学（历史）等相关专业
- **教师资格证**：初中或高中历史教师资格证均可

### 2.3 2025年成都教师公招关键信息

```
【公告来源】四川省人社厅、四川省教育厅
【招聘人数】948名（编制内正式工作人员）
【管理方式】事业单位聘用制管理
【报名时间】关注成都人事考试网官方公告
【考试时间】一般上半年3-4月、下半年10-11月
```

### 2.4 各区县招聘特点

| 区县 | 招聘特点 | 竞争程度 |
|------|---------|---------|
| 锦江区、青羊区、武侯区 | 名校多，要求高 | 高 |
| 金牛区、成华区 | 岗位较多，竞争激烈 | 中高 |
| 龙泉驿区、双流区 | 发展快，需求大 | 中等 |
| 新都区、郫都区、温江区 | 机会较多 | 中等 |
| 崇州市、金堂县、大邑县、蒲江县 | 相对容易上岸 | 较低 |

## 四、报考条件汇总

### 4.1 基本条件（通用）

- [x] 具有中华人民共和国国籍
- [x] 遵守宪法和法律，拥护中国共产党领导和社会主义制度
- [x] 具有良好的政治素质和道德品行
- [x] 身体健康，适应岗位要求的身体条件
- [x] 具备招聘岗位要求的条件和资格

### 4.2 学历与专业要求

#### 可报考历史教师的专业

| 专业代码 | 专业名称 | 适用层次 |
|---------|---------|---------|
| 045109 | 学科教学（历史） | 初中/高中 |
| 0601 | 考古学一级学科 | 高中 |
| 0602 | 中国史一级学科 | 初中/高中 |
| 0603 | 世界史一级学科 | 初中/高中 |
| 0651 | 文物一级学科 | 高中 |
| 0652 | 博物馆一级学科 | 高中 |
| 1258S1 | 文献与文化遗产保护 | 高中 |
| 040102 | 课程与教学论（历史方向） | 初中/高中 |
| 060101 | 历史学（本科） | 初中/高中 |

### 4.3 证书要求

| 证书类型 | 要求 | 获取时限 |
|---------|------|---------|
| **教师资格证** | 初中或高中历史教师资格证 | 报名时或资格审查前 |
| **普通话证书** | 二级乙等及以上 | 无明确时限 |
| **毕业证** | 国家承认学历 | 2025年7月31日前（应届生） |
| **学位证** | 相应学位 | 2025年7月31日前（应届生） |

### 4.4 年龄限制

| 岗位类型 | 年龄要求 | 放宽条件 |
|---------|---------|---------|
| 普通岗位 | 35周岁以下（1989年8月1日后出生） | 无 |
| 员额教师 | 35周岁以下 | 连续一线工作5年及以上可放宽至40周岁 |
| 高层次人才 | 视公告而定 | 副高职称等可放宽 |

### 4.5 不得报考的情形

- 曾受过各类刑事处罚的
- 曾被开除公职的
- 有违法、违纪行为正在接受审查的
- 尚未解除党纪、政纪处分的
- 按照《关于加快推进失信被执行人信用监督、警示和惩戒机制建设的意见》规定，由人民法院通过司法程序认定的失信被执行人
- 法律法规等规定的其他不能报考的情形


### 5.3 提升录取概率的策略

#### 短期策略（针对即将报考者）

1. **多地区报考**：同时关注成都、绵阳、德阳等多个地市公告
2. **灵活选择学段**：高中竞争激烈时，可考虑初中岗位
3. **关注区县机会**：二三圈层及郊区县竞争相对较小
4. **把握应届生身份**：很多岗位仅限应届生报考，竞争较小

#### 长期策略（针对在读学生）

1. **提升学历**：考取学科教学（历史）硕士研究生
2. **选择师范院校**：优先考虑部属或省属师范院校
3. **提前考证**：尽早取得教师资格证和普通话证书
4. **积累实习经验**：在学校或教育机构积累教学经验

## 六、备考建议

### 6.1 公招考试备考

#### 笔试科目

| 科目 | 内容 | 备考建议 |
|------|------|---------|
| **教育公共基础** | 教育学、心理学、教育法学、教师职业道德 | 系统学习，刷题为主 |
| **学科专业知识** | 历史学科知识、课程标准、教学设计 | 巩固专业知识，熟悉教材 |

#### 面试形式

- **试讲/说课**：10-15分钟，现场抽题或自定课题
- **结构化面试**：教育热点、应急处理、综合分析等
- **答辩**：针对试讲内容或专业问题回答

### 6.2 考核招聘备考

- **准备说课/试讲**：精心打磨3-5个课题
- **了解学校特色**：提前了解目标学校的办学理念
- **展示综合素质**：准备获奖证书、科研成果等材料

### 6.3 备考时间规划

```
【应届生备考时间线】

大三下学期（3-6月）
  └─ 确定报考方向，了解考情
  └─ 开始系统学习教育公共基础

大三暑假（7-8月）
  └─ 集中刷题，巩固知识点
  └─ 开始准备面试（试讲/说课）

大四上学期（9-12月）
  └─ 参加"蓉漂人才荟"等赴高校招聘
  └─ 关注下半年公招公告
  └─ 冲刺笔试，模拟面试

大四下学期（1-6月）
  └─ 参加上半年公招考试
  └─ 完成毕业论文，准备入职
```

## 附录：常见问题解答

### Q1：非师范专业可以报考吗？
A：可以。只要专业符合岗位要求（历史学类、学科教学等），且取得教师资格证即可报考。

### Q2：本科可以教高中吗？
A：可以。成都区县及地市州高中岗位一般要求本科及以上，但主城区高中多要求硕士。

### Q3：员额教师值得去吗？
A：员额教师虽无编制，但待遇与编制教师相近，且有机会通过考核转编，可作为过渡选择。

### Q4：成都教师待遇如何？
A：成都编制教师待遇由基本工资+绩效工资+各类补贴组成，年收入约10-15万（视区县和职称而定）。

### Q5：可以同时报考多个岗位吗？
A：同一批次考试只能报考一个岗位，但不同批次、不同地区的考试可以分别报考。
